# DS-urfu

> **Тип репозитория:** DS/governance

Планы, заметки и координация по университетским проектам (УрФУ).

## Структура

```
DS-urfu/
├── projects/
│   ├── pp-common/          # ПП — сквозные задачи (регламенты, оплата, витрина)
│   ├── pp-1/               # ПП ИРИТ-РТФ 1 курс
│   ├── pp-234/             # ПП ИРИТ-РТФ 2-4 курс
│   ├── pp-zaochniki/       # ПП заочники
│   ├── practices/          # Практики (преддипломная, учебная, проектная)
│   ├── itis-dpo/           # ИТИС ДПО
│   ├── info/               # Общие дела ИнФО
│   ├── knowledge-base/     # База знаний (pro-competencies-docs)
│   ├── ai-assistant/       # ИИ помощник (RAG, автопроверка)
│   └── practicum/          # (legacy, см. pp-common)
├── notes/                  # Заметки, рефлексии
├── processes/              # Регулярные операции, методы
└── MAPSTRATEGIC.md         # Куда движется эта система
```

## Что здесь, а что нет

| Здесь | Не здесь |
|-------|----------|
| Планы и заметки | Код (→ отдельные DS/instrument) |
| Контекст проектов | Доменное знание (→ Pack) |
| Решения, митинги | Курсы/гайды (→ DS/surface) |
