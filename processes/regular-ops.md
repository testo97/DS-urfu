# Регулярные операции

## Университет

- Новые метод-материалы по ПП / ДФ должны появляться в БЗ на Прокомпетенциях в течение 48 часов
- Собирать наработки по регламентам ПП и передавать Маркиной (отдел ПО УрФУ) для доработки → постоянное сотрудничество

## Метод работы с задачами

Из выгрузки 19 декабря 2025:

- Входящие (идеи, проекты, задачи, процессы, поручения) → регулярно разбираю по областям + минимальное описание контекста
- Проекты: карточка с описанием и целями
- Процессы: чек-листы и описания методов (можно менять, делиться, делегировать)
- Задачи: Yougile (трекинг времени, контроль мощности)
- Делегирование и контроль: Yougile (не мешается в Obsidian)
- Идеи: заметки по темам, к которым относятся

### Правила

- Имя задачи в Yougile = имя задачи/заметки в Obsidian
- Общий прогресс по проекту и следующий шаг — в Obsidian (теперь в DS-urfu)
- Регламенты описываем только для того, что делегируем или планируем делегировать
- Если не известно — не описывать, а при повторении описать
