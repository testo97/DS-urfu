### AI.1 LLM-сводка для эксперта

Сейчас эксперт читает описания проектов вручную и ищет аналоги. Нужно вспомогательное LLM-резюме (не автооценка) — в отдельной ячейке краткая сводка о наличии аналогов и готовых решений.
**Артефакт:** Скрипт/промпт, который генерирует сводку для каждой строки проекта
**Зависит от:** стабильные поля описания проекта, критерии эксперта
**Метод:** Зафиксировать формат выходной ячейки и критерии «аналога» → написать промпт
**Заметки:** Бэклог к следующему семестру. Без чёткой формы результата — сводка будет шумной
