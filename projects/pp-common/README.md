### O.1 Правила оплаты кураторов

**Артефакт:** Документ с правилами оплаты
**Подзадачи:**
- [ ] Предположить сценарии работы со студентами и список работ, важных институту
- [ ] Определить размер оплаты / стоимость часа за каждый сценарий
- [ ] Посчитать хватит ли финансов при текущем распределении
- [ ] Скорректировать если нет / много остаётся
- [ ] Показать результат НА и получить критику или одобрение

**Содержание правил:**
- Стандартный размер оплаты за студента / команду
- Размер оплаты для сложных проектов
- Размер поощрения за показатели / достижения команды
- Исключения (команда не закончила проект, получила 0 — выплата не производится)
- Лимиты выплат на одного куратора / заказчика
- Учёт условий эффективного контракта (особенно для руководителей треков)

**Наработки:**
> Базовое правило: из суммы за студентов вычесть 32% (7% фонд + 25% накладные) → разделить на 1,67 (налоги/взносы) = сумма на руки. Пример: 100 студентов × 1700 руб = 170 000 → −32% = 115 600 → ÷1,67 = **69 221 руб на руки**. Не учитывает: накладные >25%, выплаты за достижения.
